# Squiddrone_Docker

A docker environment with all necessary dependencies to build [Squiddrone project](https://github.com/elheck/Squiddrone).

Included are:
*   CMake, v3.13.4
*   GCC
*   ARM GCC in version 2019q4 major
*   doxygen with graphviz
*   lcov (for CodeCoverage)
